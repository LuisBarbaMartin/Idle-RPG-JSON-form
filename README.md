# Idle RPG JSON Form

A small React + Vite tool for building character JSON for the idle RPG data files.

## Run Locally

Install dependencies once:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Vite will print the local URL, usually `http://127.0.0.1:5173/`.

## Where To Edit

- `src/main.jsx` contains the React form, trait selection, stat pool logic, JSON preview, and download behavior.
- `style.css` contains the app styling.
- `index.html` is the Vite entry shell and usually only needs title/meta changes.
- `data/` contains the RPG JSON data files.

Avoid editing `dist/`; it is generated by `npm run build`.
